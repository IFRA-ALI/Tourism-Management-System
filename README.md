# Tourism-Management-System
The tourism management system enables user to search for the packages and book on their choice. This system will have the details of hostels, transportation, and other facilities. This system can be customized on the basis of user preferences. The system will automatically calculate the cost for each package. 

**How It Works**
You can only use the user module, enter where you want to visit, select the options; moreover, it can be customized according to your destination choice, and it is not necessary to input the actual CNIC number.
